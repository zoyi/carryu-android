A) Introduction - top

As an Android developer, you probably have a lot of problems applying a design for your application. Here we provide three themed apps. Using a flat concept for making the application look perfect. You have a wide range of custom controls in this template. Perfect for your needs.

Pre-requisites

This is a template and it is meant to help you make your app look more appealing. It is not a fully functional app. The sample projects are included to show you how to apply the design and demonstrate some useful technics. And you can use theme as a base template for your own applications. This will reduce the time to start up and make you focus more on the idea it self.

B) Features - top

  • 7 Predefined colors (Green, Blue, Lilac, Red, Orange, Purple, Yellow)
  • 6 Custom UI Controls (Edit Text, Button, Radio Button, Check Box, Seek Bar, Progress Bar)
  • 3 Themed lists
  • 3 Template Applications (List Navigation, Tab Bar Navigation, Sliding Menu Navigation)
  • Custom effects for browsing master-detail data.
  • Swipe Functionality
  • Check List with animation.
  • Load custom font from resources and apply it for all UI elements.

C) Quick Start - top

To use the theme in your code without using project templates:

  1. Copy "res" folder and past it into your project folder -be careful when pasting the files to not overwrite your own files-
  2. To attach the style to your project open "AndroidManifest.xml" and in application section change value of "android:theme" to become "@style/AppTheme", this step will remove Title bar & make background for the application "offwhite"
  3. To style control in your layout use the following code:
    style="@style/CONTROLNAME_COLORNAME"
    CONTROLNAME is one of [EditText,Button,CheckBox,RadioButton,ProgressBar,SeekBar] -Use CONTROLNAME corresponding to the control you use-
    COLORNAME is one of [Green, Blue, Lilac, Red, Orange, Purple, Yellow]
  4. To color text or back ground you can pick one of colors from colors
    android:textColor="@color/COLORNAME"
    android:background="@color/COLORNAME"

D) Usage of Sample 1 - top

This application is a skeleton application and used to make a demonstration for controls. You can use it as a base template for your project, and to do that follow the steps:

  1. Import project to your eclipse workspace
  2. Change package name, this is important especially if you want to submit the application to android store:
    • Right click on project
    • Choose Android Tools
    • Choose Rename application package
    • Type new application name then press OK
  3. Start build your own application

E) Usage of Sample 2 - top

This application is a tab application and contains a three types of themed lists. You can use it as a base template for your project, and to do that follow the steps:

  1. Import project to your eclipse workspace
  2. Change package name, this is important especially if you want to submit the application to android store:
    • Right click on project
    • Choose Android Tools
    • Choose Rename application package
    • Type new application name then press OK
  3. To add new tab:
    • Create your own fragment, you can use the "EmptyFragment" as a template
    • Open "MainActivity.java" file
    • Go to the "initialiseTabHost" function
    • Call "addTab" function and pass the required parameters
      addTab(String Resource ID, Drawable Resource ID, FRAGMENT_NAME.class, args);
    • The tab must be ready
  4. Start build your own application

F) Usage of Sample 3 - top

(note you have to import the included library for this project)

Sliding Menu App with flat design applied and contains a three types of themed lists. You can use it as a base template for your project, and to do that follow the steps:

  1. Import project to your eclipse workspace
  2. Import sliding menu library:
    • Import library included in the "Libs" folder
    • Right click the project you want it to be included into and go to properties
    • On the left side where there are tabs, select "Android"
    • In the area that says "Library", if the green check next to the library go to final step
    • If there is a red X then remove the library and press the "Add..." button
    • Select the library which you have added before
    • Do a clean build of your project
  3. Change package name, this is important especially if you want to submit the application to android store:
    • Right click on project
    • Choose Android Tools
    • Choose Rename application package
    • Type new application name then press OK
  4. To add new item to menu list:
    • Create your own fragment, you can use the "EmptyFragment" as a template
    • Open "MenuFragment.java" file
    • Go to the "initialiseMenuItems" function
    • Call "addMenuItem" function and pass the required parameters
      addMenuItem(String Resource ID, Drawable Resource ID, FRAGMENT_NAME.class, args);
    • The tab must be ready
  5. Start build your own application
