	FlatDesignSample3: Sliding Menu App with flat design applied (note you have to import the included library for this project) 

2- Documentation	

3- Fonts:
	File contains information used in this project
	
4- Libs:
	SlidingMenu: Sliding Menu Lib Which used in the third sample

5- Psds:
	PSD files that you can use to customize application icons, radiobutton or checkbox
	
6- Screen Shots
